# TeachTeam

## Prerequisites

- Ensure `npm` is installed in your machine.

## Setup Instructions

1. Clone the repository and navigate into it.
2. Run `npm install` in all four directories: 
   - `frontend`
   - `backend`
   - `admin-frontend`
   - `admin-backend`
3. Run `npm run dev` in each directory to start the localhost servers.
